# Vitegil 🔭

### 安装

执行以下命令安装依赖

```bash
pnpm i # 如果你没装过 pnpm, 可以先运行: npm install -g pnpm
```

### 开发

执行以下命令启动项目，浏览器访问 http://localhost:5173

```bash
pnpm dev
```
